16 16. Git和GitHub的区别
Git是 版本控制 软件（装在你电脑上）。
GitHub是 代码托管 网站（用浏览器访问）。
关系：Git管理版本，GitHub存放代码 + 协作。		 16. Git和GitHub的区别

17 17. 为什么用GitHub而不是网盘
网盘：只能存文件，不知道谁改了哪里。
GitHub：能看到每次修改内容、谁改的、为什么改。
协作：多人同时修改不会覆盖，可以 合并 。		 17. 为什么用GitHub而不是网盘

27 27. GitHub Pages是什么和怎么用
是什么：免费的 静态网站 托管服务。
怎么用：
1. 创建名为 `用户名.github.io` 的仓库
2. 添加HTML文件
3. 访问 `https://用户名.github.io`

用途：个人博客、项目文档、简历网站。		 27. GitHub Pages是什么和怎么用
